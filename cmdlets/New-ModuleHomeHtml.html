New-ModuleHomeHtml

Synopsis

Generates a powershell module summary page in Html.

Syntax

New-ModuleHomeHtml [-Module] <String> [-OutDir] <String> [-InProgress <Array>] [-FileName <String>] [-Header <String>] [-Footer <String>] [-Exclude <Array>] [<CommonParameters>]

Description

Uses the synopsis portion of Get-Help for a given powershell module and generates a summary page.
